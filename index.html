Selamat Datang

    Yayan He'eh (framework)- adalah framework HTML seluler gratis dan open source untuk mengembangkan aplikasi seluler hibrida atau aplikasi web dengan tampilan dan nuansa asli iOS & Android. Dengan Yayan He'eh (framework) juga dapat membuat aplikasi web prototyping yang sangat diperlukan untuk menunjukkan prototipe aplikasi web anda.

    Pendekatan utama Yayan He'eh (framework) memberi Anda kesempatan untuk membuat aplikasi web iOS & Android dengan HTML, CSS, dan JavaScript dengan mudah dan jelas. Yayan He'eh (framework) penuh dengan kebebasan.tidak membatasi imajinasi Anda Yayan He'eh (framework) benar-benar memberi Anda kebebasan!

    Yayan He'eh (framework) tidak kompatibel dengan semua platform. Yayan He'eh (framework) hanya difokuskan pada desain iOS dan Google Material untuk menghadirkan pengalaman dan kesederhanaan terbaik.

    Many More Features...

    Native Scrolling

    Terdengar aneh, tetapi salah satu fitur Yayan He'eh (framework) yang paling penting adalah ia hanya menggunakan native scrolling. Selain kelebihanya itu Anda masih dapat memiliki gulir yang mengagumkan dengan momentum dan resistensi terpental tanpa masalah gulir!

    Multiple Views (Split View)

    Yayan He'eh (framework) mendukung jumlah yang tidak terbatas dari pandangan terisolasi yang berbeda. Dan hal yang menyenangkan adalah Anda dapat dengan mudah mengontrol setiap tampilan tanpa baris JavaScript hanya menggunakan atribut "tampilan data" pada tautan.

    Library Agnostic

    Yayan He'eh (framework) tidak bergantung pada pustaka pihak ketiga. Itulah mengapa sangat ringan, kinerja sangat fleksibel.

    Clear JS API

    Tidak perlu mempelajari sesuatu yang baru dengan Yayan He'eh (framework). Memiliki metode API JavaScript yang sangat jelas dan mudah untuk mengontrol setiap bagian dari aplikasi web Anda.

    High-performance Animation

    Kinerja Yayan He'eh (framework) hanya menggunakan animasi kinerja tinggi, Dipercepat animasi dan transisi css untuk mencapai hasil terbaik.

    Pages Animation

    Salah satu target utama Yayan He'eh (framework) dalah memiliki tampilan dan perasaan asli dari aplikasi iOS dan Android. Dan Yayan He'eh (framework) dalah satu-satunya kerangka kerja yang menyelesaikannya dan menawarkan animasi halaman 1: 1 dengan efek paralaks halus, mengubah opacity dan bayangan saat memuat halaman baru.

    XHR + Caching + History + Preloading

    Ini adalah kombinasi yang tidak ada duanya untuk membuat perutean aplikasi Anda sesempurna mungkin. Framework7 siap untuk merutekan halaman aplikasi Anda menggunakan XHR (opsional) dengan metode internal sendiri dan caching internal HTML yang dapat dikonfigurasi yang memungkinkan untuk memuat halaman lebih cepat dan menghemat banyak waktu dan lalu lintas untuk pengguna Anda! Ini memuat halaman melalui Ajax satu-satunya per periode caching tertentu (10 menit secara default), kali lain hanya mendapatkannya dari memori. Ini memungkinkan untuk membuat permintaan lebih lanjut segera!

    Dom7 - Custom Dom Library

    Seperti dikatakan di atas Framework7 tidak menggunakan pustaka pihak ketiga, bahkan untuk operasi DOM, bahkan jQuery. Ini memiliki perpustakaan DOM kustom sendiri yang memanfaatkan sebagian besar metode tepi dan kinerja tinggi untuk manipulasi DOM. Anda juga dapat menggunakannya dan Anda tidak perlu mempelajari sesuatu yang baru untuk itu, ia memiliki sintaks yang sama dengan perpustakaan jQuery yang dikenal dengan dukungan metode yang paling populer dan banyak digunakan dan jQuery-like chaining.
